EXTENDED :: TILEOFFSET

Calculates the tile offset in a character block for a given number of tile blocks. 
FUNCTION TILEOFFSET(
    blocks
)
Parameters
blocks - the number of tile blocks (calculated with BLOCKS)
Return
The offset (in bytes) of a character block address to the beginning of a specific tile block.
Remarks
This can be used along with CHARBLOCK to read, write and manipulate tile data on the fly.

SRAM :: LOADBYTE

Loads the next available byte (8-bit) from static RAM. 
FUNCTION LOADBYTE
Parameters
None.
Return
Reads and returns the next byte in static RAM.
Remarks
After reading the next byte, the address pointer will self-increment.

CORE :: POKE

Writes a value to RAM. 
SUB POKE(
    address,
    n
)
Parameters
address - a 32-bit, unsigned address to write to
n - the 16-bit, halfword value to write
Return
None.
Remarks
Stores a halfword (16-bit) value into GBA RAM. Typically used with VRAM for pixel data or GBA register data.

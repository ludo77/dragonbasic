DIRECTIVES :: #TITLE

Set the title of the compiled ROM image. 
#TITLE "name"
Parameters
None.
Return
None.
Remarks
The maximum length of name is 12 characters. If it is longer, the remaining characters will be truncated. Other than that, any ASCII characters will do.

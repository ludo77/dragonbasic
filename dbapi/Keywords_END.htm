END &optional [keyword]

Description

Used to end the program or a code segment.

Results

[keyword] is used to terminate a block of code.

Valid keywords:

  • IF
  • FUNCTION
  • SELECT
  • SUB

Example

; create a function that will set the graphics mode
function init
    graphics 0,true
end function

; create a simple program
start:
    if x = 1
        ; do something
    end if

    select x
    case 1
        ; do something
    case 2
        ; do something
    default
        ; do something
    end select

while

loop   ; end program

DEFAULT ...

Description

SELECT … END SELECT clause to execute if all other CASE statements fail.

Results

When used inside a SELECT … END SELECT clause, … following DEFAULT will execute only if all other CASE [expr] statements fail.

Example

; see what level we want to load
select level
case 1
    addr_ptr = level1!
case 2
    addr_ptr = level2!
default ; any other level
    level_not_made_yet
end select

TILES :: COLORTILE

Set the palette that a tile should use. 
SUB COLORTILE(
    address,
    index
)
Parameters
address - an unsigned, 32-bit address returned by TILE
index - the palette index to use from BG_PALETTE
Return
None.
Remarks
Sets the new palette for a particular tile to use when being drawn by the GBA. The background must be in 16-color mode for this routine to have any effect.

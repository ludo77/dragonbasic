TILES :: UPDATETILES

Update background scroll and rotation information. 
SUB UPDATETILES(
    bg
)
Parameters
bg - the background layer (0-3)
Return
None.
Remarks
After scrolling or rotating a background, this subroutine must be called to copy this information to the GBA hardware registers. This should only be done during a vertical blank.

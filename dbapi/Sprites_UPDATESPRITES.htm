SPRITES :: UPDATESPRITES

Updates all OAM data. 
SUB UPDATESPRITES
Parameters
None.
Return
None.
Remarks
Copies all data in RAM (duplicated OAM) to OAM memory to update the GBA hardware. This should only be done during a vertical blank period.

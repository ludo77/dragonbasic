FUNCTION name &optional ([arg1,arg2,...])

Description

Declares a new function.

Results

Creates a new function in ROM with name. Optionally, parameters can be passed to the function to make use of.

Example

; create a function to sum a list of numbers
function sum_array(list,count)
    total = 0 ; <-- total is global
    for x = 0 to count - 1
        total = total + list[x]
    next
    return total
end function

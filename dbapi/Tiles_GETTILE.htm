TILES :: GETTILE

Gets the tile used by an address. 
FUNCTION GETTILE(
    address
)
Parameters
address - an unsigned, 32-bit address returned by TILE
Return
The tile index used by a particular tile (all bits except palette and flip data).
Remarks
This function is commonly used to check if a tile is mapped in a certain way (to check for collision information, etc.)

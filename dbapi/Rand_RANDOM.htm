RAND :: RANDOM

Computes a pseudo-random number in a range. 
FUNCTION RANDOM(
    min,
    max
)
Parameters
min - the smallest value to return
max - the largest value to return
Return
Returns a random number in the range min-max.
Remarks
The value min can be returned, but the largest return value in max-1.

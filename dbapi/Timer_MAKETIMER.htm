MAKETIMER freq

Initializes the user-defined timer.

Arguments

Parameter Meaning
freq the number of times the timer will fire per second

Results

Creates the new timer with all appropriate parameters. Note: the user-defined timer uses both timers 2 & 3 of the GBA hardware.

Example

; create a new timer that fires 10 times a second
maketimer 10

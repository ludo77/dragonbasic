PALETTE :: SETPALENTRY

Sets the 15-bit, RGB color value of a palette entry. 
SUB SETPALENTRY(
    palette,
    index,
    entry,
    color
)
Parameters
palette - either BG_PALETTE (&5000000) or SPRITE_PALETTE (&5000200)
index - the 16-color palette index to rotate (0-15)
entry - the actual color entry to set (0-15)
color - a 15-bit, RGB color value
Return
None.
Remarks
Sets the color value of an entry in the 16-color palette defined by palette and index.

SPRITES :: POSITIONSPRITE

Set the absolute position of a sprite 
SUB POSITIONSPRITE(
    n,
    x,
    y
)
Parameters
n - the sprite index (0-127)
x - the x-coordinate of the position (0-511)
y - the y-coordinate of the position (0-255)
Return
None.
Remarks
The x and y coordinates can be negative (to be offscreen). But cannot be larger than (511,255) respectively.

BITMAPPED Subroutines & Functions

BLIT - Copies a bitmap image onto the screen with a mask.
CLS - Fills the entire screen with a solid color.
FLIP - Toggles the currently visible buffer in a double-buffered mode.
LINE - Draws a line on the screen.
PIXEL - Gets the color of a pixel
PLOT - Plots a pixel on the screen
SCREEN - Returns the currently hidden buffer in a double-buffer mode.
WALLPAPER - Copies a fullscreen image from ROM to VRAM.

EXTENDED :: BLOCKS

Calculates how many tile blocks an image uses. 
FUNCTION BLOCKS(
    width,
    height,
    depth
)
Parameters
width - width of the image in pixels (must be a multiple of 8)
height - height of the image in pixels (must be a multiple of 8)
depth - the bit-depth of the image (4 or 8)
Return
The number of 8x8, 4-bit tile blocks needed to hold this image in RAM on the GBA.

INTERRUPT name

Description

Creates a new interrupt service routine.

Results

Defines name as the label pointing to a new ISR. It can then be used with any of the ON… functions. Interrupt routines should be extremely short and fast! If an interrupt handler is too long, your program may begin to experience problems. Note: INTERRUPT routines must be matched with EXIT.

Example

; create a new interrupt routine
interrupt refresh
    updatesprites
    exit

start:
    ; set refresh to be called automatically every vblank
    onvblank refresh

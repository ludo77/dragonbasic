STRINGS :: LEFT$

Gets the left portion of a string. 
FUNCTION LEFT$(
    string$,
    length
)
Parameters
string$ - the string to be parsed
length - the length of the returned string
Return
A new string which is the first 'length' characters of string$.
Remarks
If length is less than or equal to 0, the result is undefined.

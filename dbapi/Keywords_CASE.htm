CASE [expr] ...

Description

Compares the current SELECT expression against [expr].

Results

Tests the current SELECT expression against [expr]. If they match (are equal) then … is executed. Once … finishes, execution will continue after END SELECT.

Example

; act based on what buttons are pressed
select not( keys )
case key_a or key_b
    ; a and b both pressed
case key_start
    ; just start button
default
    ; any other combination
end select

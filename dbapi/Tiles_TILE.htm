TILES :: TILE

Get the address of a specific tile. 
FUNCTION TILE(
    screen,
    x,
    y
)
Parameters
screen - the screen block to index into (0-31)
x - the x-coordinate of the tile (0-31)
y - the y-coordinate of the tile (0-31)
Return
An unsigned, 32-bit address of the tile.
Remarks
While background layers can be more that 1 screen block in size, this function only indexes into a single screen block.

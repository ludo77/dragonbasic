SPRITES :: SIZESPRITE

Sets the hardware drawing size of a sprite. 
SUB SIZESPRITE(
    n,
    shape,
    size
)
Parameters
n - the sprite index (0-127)
shape - the shape of the sprite (SQUARE, WIDE or TALL)
size - the size of the sprite (SIZE_8, SIZE_16, SIZE_32 or SIZE_64)
Return
None.

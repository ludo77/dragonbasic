RAND :: RND

Computes a 15-bit, pseudo-random number. 
FUNCTION RND
Parameters
None.
Return
A 15-bit, pseudo-random number from &0-&7FFF.
Remarks
Random numbers for a given SEED will always be the same. RND can be used to create random RGB colors well. For a random fixed-point value, just NAND with &FF for a value ranging from 0-1.0.

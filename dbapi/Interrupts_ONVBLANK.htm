INTERRUPTS :: ONVBLANK

Creates or halts a vertical blank interrupt control. 
SUB ONVBLANK(
    i
)
Parameters
i - the interrupt to be called or 0 to disable
Return
None.
Remarks
This interrupt will fire each time the display enters another vertical blank period. This is good for timing 1/60th of a second precise speed and handling game graphics updates.

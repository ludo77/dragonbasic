STRINGS :: MID$

Gets the middle portion of a string. 
FUNCTION MID$(
    string$,
    start,
    length
)
Parameters
string$ - the string to be parsed
start - the initial index to begin parsing at
length - the length of the returned string
Return
A new string which starts at 'start' and is 'length' characters of string$.
Remarks
If length is less than or equal to 0, the result is undefined. If the index is negative or greater than the length of the original string, then the result is undefined.

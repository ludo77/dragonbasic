INPUT :: WAITKEY

Waits for a button to be pressed and released. 
FUNCTION WAITKEY(
    mask
)
Parameters
mask - a bit-mask of all buttons to test for, OR'ed together with NOR
Return
Returns the value of the button in mask that was pressed and then released.
Remarks
WAITKEY is perfect for menu selections. It allows you to wait for multiple buttons, but only return one. It also will wait for the button to be released before returning.

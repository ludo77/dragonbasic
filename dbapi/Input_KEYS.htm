INPUT :: KEYS

Returns the current input state. 
FUNCTION KEYS
Parameters
None.
Return
Reads REG_P1 from the GBA hardware and returns it. All bits are flipped, so that a 0 is not-pressed and 1 is pressed.
Remarks
The return value can be masked using an AND operation along with a KEY_ constant to see if a button is pressed.

STRINGS :: PRINT

Prints a string to a tiled background. 
SUB PRINT(
    tile,
    string$
)
Parameters
tile - the address of the tile to begin printing at
string$ - the string to print
Return
None.
Remarks
Can only be used in a tiled mode, and must have a font loaded. It will only print 8x8 tiles in sequential order.

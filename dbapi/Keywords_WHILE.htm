WHILE &optional [expr]

Description

Begins a WHILE … LOOP clause.

Results

Begins a loop to be terminated with LOOP. If [expr] is present, the loop will only execute while [expr] is true; [expr] is evaluated every iteration of the loop. If [expr] is not present, an infinite loop is created.

Example

; create an infinite game loop
while
    ; inside the game loop, create a loop that happens
    ; while the player is pressing the -B- button
    while key(key_b)
        ; do something
    loop
loop

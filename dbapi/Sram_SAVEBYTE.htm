SRAM :: SAVEBYTE

Saves a byte (8-bit) to static RAM. 
SUB SAVEBYTE(
    x
)
Parameters
x - the byte to save
Return
None.
Remarks
This subroutine can only be used to save bytes to static RAM (&E000000). It will not work on other RAM addresses.

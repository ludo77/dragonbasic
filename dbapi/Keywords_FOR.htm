FOR var=[expr] to|downto [goal] &optional step [n]

Description

Begins a FOR loop.

Results

Compiles a FOR loop. Sets the counter variable 'var' to [expr]. If the keyword TO is used, then the loop continues executing until var > [goal]. If DOWNTO is used, execution continues until va < [goal].

An optional STEP parameter is allowed. If specified, then var is incremented by [n] each iteration. Otherwise, [n] defaults to 1 for TO and -1 for DOWNTO.

Example

; loop through all the indices of a loop, skipping every other one
for x = 0 to 99 step 2
    ; do something with list[x]

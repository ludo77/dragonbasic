INPUT :: KEYUP

Returns a value indicating whether any of the specified keys have been released between the last two calls to CHECKKEYS. 
FUNCTION KEYUP(
    mask
)
Parameters
mask - a bit-mask of all buttons to test for, OR'ed together.
Return
Returns non-zero if any of the keys specified in the mask have been released between the last two calls to CHECKKEYS.
Remarks
Use this function to check for one or more specific buttons transitioning from pressed to released state without having to keep track of their current states.
A non-zero value indicates the button(s) is/are pressed and 0 is released.

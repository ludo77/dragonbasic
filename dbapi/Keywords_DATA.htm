DATA n1 &optional ,n2 &optional ...

Description

DATA can be used to write constants into ROM.

Results

Compiles n1,n2,… into the ROM with 32-bit data.each value.

Example


; create some data
my_data:
    data 1,2,3,4,5



; create an array
dim x(5)

start:
    ; restore and read the data
    restore my_data
    for k = 0 to 5
        read x[k]
    next

PALETTE :: RGB

Creates a 15-bit, RGB color value from components. 
FUNCTION RGB(
    r,
    g,
    b
)
Parameters
r - red component of the RGB value (0-31)
g - green component of the RGB value (0-31)
b - blue component of the RGB value (0-31)
Return
A 15-bit, RGB color value.
Remarks
Colors created with RGB can be used to create palettes and set entries in palettes as well as plot pixels, lines, etc. in bitmapped modes 3 and 5.

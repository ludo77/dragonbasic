SRAM Subroutines & Functions

LOADBYTE - Loads the next available byte (8-bit) from static RAM.
LOADLONG - Loads the next available word (32-bit) from static RAM.
LOADTEXT$ - Loads the next available counted string frm static RAM.
LOADWORD - Loads the next available halfword (16-bit) from static RAM.
SAVEBYTE - Saves a byte (8-bit) to static RAM.
SAVELONG - Saves a word (32-bit) to static RAM.
SAVETEXT - Saves a counted string to static RAM.
SAVEWORD - Saves a halfword (16-bit) to static RAM.

RESTORE

Description

The data pointer is set by RESTORE.

Results

Once the data pointer has been set with RESTORE, data may be READ from the ROM into variables.

Example

; create some data
my_data:
    data 1,2,3,4,5


; create an array
dim x(5)

start:
    ; restore and read the data
    restore my_data
    for k = 0 to 5
        read x[k]
    next

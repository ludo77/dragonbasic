SRAM :: SAVEWORD

Saves a halfword (16-bit) to static RAM. 
SUB SAVEWORD(
    x
)
Parameters
x - the halfword to save
Return
None.
Remarks
This subroutine can only be used to save data to static RAM (&E000000). It will not work on other RAM addresses.

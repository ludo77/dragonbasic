TILES Subroutines & Functions

CHARBLOCK - Gets the address of a character block.
CLEARTILES - Erase an area of tiles.
COLORTILE - Set the palette that a tile should use.
DISABLETILES - Disable a background layer.
ENABLETILES - Enables a background layer.
FLIPTILE - Flips or mirrors a tile.
GETTILE - Gets the tile used by an address.
LOADFONT16 - Copies a 16-color font into VRAM.
LOADFONT256 - Copies a 256-color font into VRAM.
LOADTILES - Load a character block with tile data.
MAPIMAGE - Copy a map from ROM to a screen block.
MAPTILES - Copy a map from ROM to a screen block.
ORDERTILES - Set the z-order of a background layer.
SCREENBLOCK - Gets the address of a screen block.
SCROLL - Scrolls a background layer.
SCROLLPOS - Scrolls a background layer.
SCROLLX - Get the number of pixels scrolled.
SCROLLY - Get the number of pixels scrolled.
SETTILE - Set the tile of a given address.
TILE - Get the address of a specific tile.
UPDATETILES - Update background scroll and rotation information.

ROTATIONS :: MAKEROTATION

Setup one of the 4 available rotation matrices. 
SUB MAKEROTATION(
    matrix,
    sx#,
    sy#,
    angle
)
Parameters
matrix - a rotation matrix index (0-3)
sx# - the horizontal scaling factor
sy# - the vertical scaling factor
angle - an angle between 0-359
Return
None.
Remarks
Computes all the values necessary to create a sprite rotation matrix. Scaling factors of 1.0# are normal, 0.5# is double sized and 2.0# is scaling of 1/2 size.

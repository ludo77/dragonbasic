DIRECTIVES :: #BITMAP

Import a BMP or PCX file. 
#BITMAP "file"
Parameters
None.
Return
None.
Remarks
If the BMP or PCX image is 16- or 256-colors, the compiler will break it into 8x8 pixel tiles, left->right, top->bottom as is needed for the GBA LOADTILES and LOADSPRITE functions. A 24-bit image (true color) will be converted to 15-bit RGB format for the GBA bitmapped modes.

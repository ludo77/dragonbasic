INTERRUPTS :: DISABLEINTERRUPTS

Turns off all interrupts. 
SUB DISABLEINTERRUPTS
Parameters
None.
Return
None.
Remarks
This will disable all interrupts immediately. NOTE: PLAYMUSIC and PLAYSOUND will terminate if this is called!

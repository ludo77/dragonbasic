PALETTE :: LOADPAL16

Loads a 16-color palette into RAM. 
SUB LOADPAL16(
    palette,
    index,
    label
)
Parameters
palette - either BG_PALETTE (&5000000) or SPRITE_PALETTE (&5000200)
index - the 16-color palette index to load (0-15)
label - address of the palette imported with #PALETTE
Return
None.
Remarks
Copies 16, 15-bit, RGB colors from label into the 16-color palette defined by palette and index.

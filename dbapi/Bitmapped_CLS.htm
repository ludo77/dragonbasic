BITMAPPED :: CLS

Fills the entire screen with a solid color. 
SUB CLS(
    color
)
Parameters
color - a 15-bit, RGB color or a palette index
Return
None.
Remarks
Only works in bitmapped modes (3, 4 and 5). In modes 3 and 5, color is a 15-bit, RGB value. In mode 4, color must be an index into the palette.

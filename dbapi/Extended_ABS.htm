EXTENDED :: ABS

Calculates the absolute value of an integer. 
FUNCTION ABS(
    n
)
Parameters
n - a signed integer
Return
The absolute value of n ( |n| ).
Remarks
Checks to see if n is negative, and if it is, performs a two's complement and returns the new value.

MAKETIMER ticks

Waits until the timer has fired so many times.

waittimer

Parameter Meaning
ticks the number of fires to wait for

Results

Halts program execution (except interrupts) until CLOCKTIMER returns >= ticks. This will not necessarily be the total number of ticks waited unless preceeded with a call to RESETTIMER.

Example

; create a timer and wait for 3 seconds
maketimer 1
starttimer
waittimer 3

SPRITES :: MAKESPRITE

Setup a new sprite object. 
SUB MAKESPRITE(
    n,
    char
)
Parameters
n - the sprite index (0-127)
char - the sprite character data to use (0-1023)
Return
None.
Remarks
Creates a new, 256-color sprite to begin manipulating. The sprite will use the graphical data found at char.

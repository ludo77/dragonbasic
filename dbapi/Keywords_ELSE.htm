ELSE ...

Description

Used inside an IF … END IF clause to denote an optional execution path.

Results

When used inside an IF [expr] … END IF clause, … following ELSE will execute if [expr] returns false (0).

 

Example

; checks to see if a sprite is on the right half of the screen
if spritex(my_sprite) > 120
    ; execute code when on right
else
    ; execute code when on left
end if

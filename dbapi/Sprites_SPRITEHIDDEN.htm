SPRITES :: SPRITEHIDDEN

Check to see if a sprite is onscreen. 
FUNCTION SPRITEHIDDEN(
    n
)
Parameters
n - the sprite index (0-127)
Return
Non-zero if the sprite is offscreen, otherwise 0.
Remarks
If a sprite is to the left or above the viewport, but is big enough to still be partially visible, this function will return true.

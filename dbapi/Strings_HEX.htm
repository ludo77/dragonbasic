STRINGS :: HEX$

Converts an integer to a string. 
FUNCTION HEX$(
    n
)
Parameters
n - the unsigned integer to convert
Return
A string that is the hexadecimal representation of the number n.
Remarks
The number is converted using base 16, and it is treated as unsigned (no negative values).

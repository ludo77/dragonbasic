INPUT :: CHECKKEYS

Returns the current input state. 
SUB CHECKKEYS
Parameters
None.
Remarks
CHECKKEYS keeps track of transitions in the state of keys. After calling CHECKKEYS it is possible to use KEYUP and KEYDOWN to determine which keys have changed state since the last call to CHECKKEYS.

TILES :: ENABLETILES

Enables a background layer. 
SUB ENABLETILES(
    bg,
    screen,
    char,
    flags
)
Parameters
bg - the background layer to enable (0-3)
screen - the screen block to use for mapping (0-31)
char - the character block to use for tile data (0-3)
flags - a bit-mask of all flags to set, OR'ed together with NOR
Return
None.
Remarks
Enables the background layer bg. If flags is 0, then the default size of 256x256 is used and the background is 16 colors.

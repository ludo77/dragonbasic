PALETTE :: ROTATEPAL256

Rotates all the colors in a palette. 
SUB ROTATEPAL256(
    palette
)
Parameters
palette - either BG_PALETTE (&5000000) or SPRITE_PALETTE (&5000200)
Return
None.
Remarks
Rotates the upper 255-color entries in the 256-color palette defined by palette, leaving the transparency color alone.

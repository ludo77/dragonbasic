TILES :: LOADTILES

Load a character block with tile data. 
SUB LOADTILES(
    char,
    offset,
    label,
    nblocks
)
Parameters
char - the character block to load into (0-3)
offset - the tile offset (in blocks) to begin loading at
label - the source label of tile data imported with #BITMAP
nblocks - the number of blocks used by the imported tiles
Return
None.
Remarks
Copies tile data from ROM to VRAM. This data is the image data to be drawn by a screen block (map data).

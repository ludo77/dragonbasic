SELECT [expr]

Description

Begins a SELECT clause where [expr] is tested against many possible CASE expressions.

Results

Compiles a SELECT expression, [expr], to be tested against various CASE expressions. If none of them match, a DEFAULT block of code will execute.

Example

; act based on what buttons are pressed
select not keys
case key_a or key_b
    ; a and b both pressed
case key_start
    ; just start button
default
    ; any other combination
end select

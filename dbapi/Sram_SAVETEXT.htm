SRAM :: SAVETEXT

Saves a counted string to static RAM. 
SUB SAVETEXT(
    string$
)
Parameters
string$ - the string to save
Return
None.
Remarks
This subroutine can only be used to save a string to static RAM (&E000000). It will not work on other RAM addresses.

SPRITES :: BUMPSPRITES

Checks to see if two sprites overlap. 
FUNCTION BUMPSPRITES(
    n1,
    n2
)
Parameters
n1 - the sprite to check (0-127)
n2 - the sprite to check against (0-127)
Return
Returns 0 if they do not overlap, otherwise non-zero.
Remarks
This is only a bounding box check, and is not pixel-perfect.

Dragon BASIC, online at http://github.com/uli/dragonbasic

EXTENDED Subroutines & Functions

ABS - Calculates the absolute value of an integer.
ALIGNED - Calculates an aligned address.
ALLOCATE - Dynamically allocates an array of bytes.
BLOCKS - Calculates how many tile blocks an image uses.
DMACOPY - Copies words (32-bit) of data from one address to another.
DMAWAIT - Waits for the current DMA transfer to finish.
FIX# - Converts an integer to a fixed-point value.
FSQRT# - Calculates the square root value of a fixed-point value.
INT - Converts a fixed-point value to an integer.
ROUND - Rounds a fixed-point value up or down and returns an integer.
SQRT - Calculates the square root value of an integer.
TILEOFFSET - Calculates the tile offset in a character block for a given number of tile blocks.

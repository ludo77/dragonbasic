BITMAPPED :: LINE

Draws a line on the screen. 
SUB LINE(
    x1,
    y1,
    x2,
    y2,
    color
)
Parameters
x1 - the x coordinate of the start pixel
y1 - the y coordinate of the start pixel
x2 - the x coordinate of the finish pixel
y2 - the y coordinate of the finish pixel
color - a 15-bit, RGB value or palette index
Return
None.
Remarks
Draws a line on the screen. In mode 4, color must be a palette index. No clipping is performed, and lines moving offscreen will wrap.

STRINGS :: MOVE

Copies one string into another. 
SUB MOVE(
    dest$,
    source$
)
Parameters
dest$ - the destination string
source$ - the string to copy
Return
None.
Remarks
All 256 bytes of the string is copied, regardless of length. This is mainly used internally by TIN, as in Dragon BASIC you can use the = operator to set strings.

PALETTE :: RGBR

Gets the red component from a 15-bit, RGB color value. 
FUNCTION RGBR(
    color
)
Parameters
color - a 15-bit, RGB color value
Return
The 5-bit, red component of color.
Remarks
The component is masked and shifted down to the correct level for a value in the range of 0-31.

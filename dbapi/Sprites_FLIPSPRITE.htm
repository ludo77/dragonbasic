SPRITES :: FLIPSPRITE

Flips or mirror a sprite when drawn. 
SUB FLIPSPRITE(
    n,
    x,
    y
)
Parameters
n - the sprite index (0-127)
x - flag to mirror the sprite about the y-axis (horizontally).
y - flag to flip the sprite about the x-axis (vertically).
Return
None.
Remarks
x and y should be 0 to disable the appropriate flip/mirror bits and non-0 to enable them. If multiple sprites use the same sprite character, flipping one will not flip the other.

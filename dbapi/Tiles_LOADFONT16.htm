TILES :: LOADFONT16

Copies a 16-color font into VRAM. 
SUB LOADFONT16(
    char,
    label
)
Parameters
char - the character block to load into (0-3)
label - the source label of tile data imported with #BITMAP
Return
None.
Remarks
All 16-color fonts are loaded into tile offset 32 and are 96 blocks in size. The image should be 768x8 (or 8x768) in size, 16-colors, and consist of the ANSI characters 0x20-0x7F in order.

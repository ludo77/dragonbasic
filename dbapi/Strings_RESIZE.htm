STRINGS :: RESIZE

Resizes the length of a string. 
SUB RESIZE(
    string$,
    length
)
Parameters
string$ - the string to resize
length - the new length of the string
Return
None.
Remarks
Sets the new length of a string. The is mainly used for the internal use of Dragon BASIC, but can be used instead of LEFT$ to snag the left portion of a string. If the length is increased, the superfluous bytes are undefined.

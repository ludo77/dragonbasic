PALETTE Subroutines & Functions

GETPALENTRY - Gets the 15-bit, RGB color value of a palette entry.
LOADPAL16 - Loads a 16-color palette into RAM.
LOADPAL256 - Loads a 256-color palette into RAM.
MAKEPALETTE - Creates a web-safe, 216-color palette in RAM.
RGB - Creates a 15-bit, RGB color value from components.
RGBB - Gets the blue component from a 15-bit, RGB color value.
RGBG - Gets the green component from a 15-bit, RGB color value.
RGBR - Gets the red component from a 15-bit, RGB color value.
ROTATEPAL16 - Rotates all the colors in a palette.
ROTATEPAL256 - Rotates all the colors in a palette.
SETPALENTRY - Sets the 15-bit, RGB color value of a palette entry.

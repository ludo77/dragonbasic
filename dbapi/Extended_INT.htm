EXTENDED :: INT

Converts a fixed-point value to an integer. 
FUNCTION INT(
    f#
)
Parameters
f# - a fixed-point value
Return
The whole portion of f# without the fraction.
Remarks
Converts f# to an integer value by bit-shifting right the whole value.

RETURN &optional [expr]

Description

RETURN is used from within a subroutine or FUNCTION to exit either. When used inside a function, it has the option of returning a value for the function.

Results

When used inside a subroutine, it halts execution of the subroutine and returns to the subroutine that called it (with GOSUB).

When used inside a FUNCTION, it can optionally return a value ([expr]).

Example

; create a subroutine to update all sprites
update:
    vblank
    updatesprites
    return

; create a function to return the sum of two numbers
function add(n1,n2)
    return n1+n2
end function

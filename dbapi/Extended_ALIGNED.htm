EXTENDED :: ALIGNED

Calculates an aligned address. 
FUNCTION ALIGNED(
    address
)
Parameters
address - a 32-bit, unsigned address
Return
A word aligned, 32-bit, unsigned address.
Remarks
If address is already word aligned, ALIGNED will return address. Otherwise, it will return the next available aligned address.

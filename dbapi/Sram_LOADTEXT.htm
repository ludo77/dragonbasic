SRAM :: LOADTEXT$

Loads the next available counted string frm static RAM. 
FUNCTION LOADTEXT$
Parameters
None.
Return
A counted string.
Remarks
The counted string is copied into the destination. Increments the address pointer by the length of the string + 1.

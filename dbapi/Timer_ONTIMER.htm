INTERRUPTS :: ONTIMER

Creates or halts a user-defined, timer interrupt. 
SUB ONTIMER(
    i
)
Parameters
i - the interrupt to be called or 0 to disable
Return
None.
Remarks
This interrupt will fire each time the current user-defined timer fires. Very good for controlling enemy movement in a game.

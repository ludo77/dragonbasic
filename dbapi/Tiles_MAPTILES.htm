TILES :: MAPTILES

Copy a map from ROM to a screen block. 
SUB MAPTILES(
    address,
    label,
    width,
    height
)
Parameters
address - an unsigned, 32-bit address returned by TILE
label - the address label of data created with MAP
width - the number of tiles horizontally to map
height - the number of tiles vertically to map
Return
None.
Remarks
Copies all the map data at label to the tile beginning at address and bounded by width and height. Only the tile and flip data is copied! No palette information is written.

BITMAPPED :: PLOT

Plots a pixel on the screen 
SUB PLOT(
    x,
    y,
    color
)
Parameters
x - the x coordinate of the pixel
y - the y coordinate of the pixel
color - a 15-bit, RGB value or palette index
Return
None.
Remarks
In modes 3 and 4, the resolution is 240x160. In mode 5 is it 160x120. In modes 3 and 5, color is a 15-bit, RGB value. In mode 4, color is a palette index.

EFFECTS :: MOSAIC

Sets the mosaic registers in the GBA. 
SUB MOSAIC(
    bgx,
    bgy,
    objx,
    objy
)
Parameters
bgx - background, horizontal scaling factor (0-15)
bgy - background, vertical scaling factor (0-15)
objx - sprites, horizontal scaling factor (0-15)
objy - sprites, vertical scaling factor (0-15)
Return
None.
Remarks
Sets the scaling factor for all background layers and sprites. 0 is off (normal view) and 15 is the maxmimum pixelation.

STOPTIMER

Stops the user-defined timer from firing.

Arguments

No arguments

Results

Halts the user-defined timer. Note: this will not reset the timer to 0. You must use RESETTIMER to do this.

Example

; create an interrupt for the timer
interrupt my_timer_int
    stoptimer ; stop the timer
    exit

start:
    ; create a timer and let it go
    maketimer 10
    starttimer

    ; in 1/10 of a second it will stop automatically…

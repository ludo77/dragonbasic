TILES :: SCREENBLOCK

Gets the address of a screen block. 
FUNCTION SCREENBLOCK(
    n
)
Parameters
n - a screen base block (0-31)
Return
The address of a screen base block in VRAM.
Remarks
Screen blocks contain map data. Each index in a screen block is a pointer to a tile in a corresponding character base block.

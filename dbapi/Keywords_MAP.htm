MAP n1 &optional ,n2 &optional ...

Description

MAP can be used to write constants into ROM.

Results

Compiles n1,n2,… into the ROM with 16-bit data each value.

Example

; create some data
my_map:
    map 1,2,3,4,5,6  ; must be even before other code
    map 1,2,3,4,5,6  ; here we have 12 so it's ok.

start:
    ; some code here then
   maptiles tile(8,0,0),my_map,6,2
